# Terraform Multi-AZ VPC Infrastructure

Production-grade AWS networking infrastructure built using Terraform.

## Overview-----------------------------------

This project provisions a highly available Multi-AZ VPC architecture on AWS.

The design includes:

- Public Subnets
- Private Subnets
- Internet Gateway
- NAT Gateway
- Route Tables
- Security Groups
- Multi-Environment Support

## Technologies----------------------------------------

- AWS VPC
- Terraform
- EC2
- NAT Gateway
- Internet Gateway

## Architecture

VPC → Public Subnets → NAT Gateway

VPC → Private Subnets → Application Resources

## Features------------------------------------------------------

- Infrastructure as Code
- Multi-AZ Design
- Secure Network Segmentation
- High Availability
- Reusable Terraform Modules
